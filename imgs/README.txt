#####################################################################
# Description
#####################################################################

This directory currently contains only images captured by a Nikon
D7000 camera using the normal setting and the Fl (flourescent) 
color transform. Three brightness levels are provided, more
can be found online. NEF files are raw, JPG files are processed by
the camera, and the tiff files are the NEF files converted to tif 
format to be more easily read by Matlab. This conversion was done 
with a script contained within the src/Matlab/ directory.
